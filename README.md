# Python Assistant Chatbot

A Python-based AI chatbot that acts as a Python mentor.
Built using:

- FastAPI (Backend)
- Ollama (LLM Integration)
- HTML/CSS/JS (Frontend)
- Markdown rendering
- Code highlighting
- Copy button for code blocks

## Features

- Answer Python questions
- Step-by-step explanations
- Code formatting
- Syntax highlighting
- Copy-to-clipboard
- REST API architecture

## How To Run

1. Install dependencies
2. Start Ollama
3. Run FastAPI server
4. Open index.html
